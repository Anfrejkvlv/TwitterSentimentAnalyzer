# TwitterSentimentAnalyzer
TwitterSentimentAnalyzer est un projet développé avec Spring AI visant à analyser 
automatiquement les sentiments exprimés sur Twitter (X) et traités à l’aide de techniques de traitement du langage naturel (NLP) et
d’intelligence artificielle grâce à la bibliothèque Stanford CoreNLP, puis classifié selon son sentiment : VERY POSITIVE, POSITIVE, NEGATIVE, NEUTRAL, VERY NEGATIVE à propos de Microsoft et 
de ses produits (Windows, Azure, Office, Xbox, etc.).
# Source de données 
[Lien Kaggle:] [https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis?resource=download]
